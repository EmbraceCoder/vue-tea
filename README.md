# vue-tea
## 一. 创建仓库

1.1 克隆代码
1.2 在 `master` 下创建 `develop` 分支, 并提交到远程仓库中

```git

    git branch <分支名>
    
    git push --set-upstream origin <分支名称>

```
1.3 在 `develop` 下创建功能分支


## 二. 初始化项目

2.1 创建前端项目
```git
    vue create <项目名称>
```

2.2 创建后端项目
```git
    npm install express-generator -g
    express --view=ejs <项目名称>
```
